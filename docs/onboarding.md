# Guía de incorporación

## Objetivo
Este repositorio contiene el servicio de OTP para autenticación de segundo factor.

## Inicio rápido
1. Abre el repositorio en VS Code.
2. Ejecuta Dev Containers: Reopen in Container.
3. Espera el mensaje de bootstrap completado.
4. Inicia el servicio:

```bash
./mvnw spring-boot:run
```

5. Verifica disponibilidad en puerto 8081.
